## Get Up and Running

```
npm install
npm run build
```

### Set up Enviroment

-   Configure ports
-   Get a PostGres Database / Get Connection Info
-   fill .env with your database info (Database, Port, ect...)
-   (see ExampleEnv.txt for reference)

apply the database schema

```
npm run apply-schema
```

### Start

```
npm start
```
